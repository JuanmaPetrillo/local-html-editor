# Technology Stack

## Current implementation (now)

Current state is a static browser-first prototype:

- Static HTML shell (`apps/desktop/index.html`)
- Vanilla JavaScript modules (`.mjs`) in `apps/desktop/src/`
- Node-based local validation scripts in `scripts/*.mjs`

This is the active implementation for Milestones 0-2A.

## Target stack (later milestones)

Target architecture remains:

- Tauri 2 desktop shell
- React + TypeScript + Vite frontend

Important: this production target is **not** the current code state.
